### 项目运行方式:

>1.  直接运行 `main.exe` 即可



>2. 打开 main.py

首先安装 PyQT5

```bash
pip install PyQt5
```

然后进入 `\src` 目录，运行 `main.py` 文件

```bash
python main.py
```

既可以看见运行界面
